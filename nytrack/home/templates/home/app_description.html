Project Description

    This Project intends to demonstrate our abilities to communicate, query and integrate a database by mimicking the Amtrak railway service. The source code as well as the documentation for this project can be found here. The following are some small details to consider:

  • All ticket bookings are only for the month of January 2018.
  • All dates are in the format: 2018-08-01 6:00:00

Instructions

    To Check for the Train Time Tables or Tickets purchased simply navigate to Display Train Schedule. In order to purchase a ticket simply navigate to Buy Or Search

    Purchase Ticket

      Upon clicking Purchase button, user is prompted with:

      • Departure Station
      • Destination Station

      Once the user has selected the stations, a list of available times is generated for the user to select. Once again the user must choose the stations and is prompted to choose:

      • Departure Date
      • Method of Payment
      • NOTE: Trip information is saved on database
